# Port Scanner

## Built with
- Python
- Socket library
- User input
- Functions
- Error handling (`try` / `except`)
- Dictionaries
- Loops
- Type casting

## Features
- Scan TCP ports on a target host
- Detect open ports
- Custom port range selection
- User-defined scan target
- Service identification for common ports

## Concepts learned
- TCP networking
- IPv4 addressing
- Socket creation and connection handling
- Exception handling
- Function design
- User input validation
- Basic network reconnaissance

## Improvements
1. Improve service detection
2. Implement multithreading
3. Add timeout configuration
